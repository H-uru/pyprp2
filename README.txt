==NOTES==
* Currently, this is only compatible with 32-bit Blender. You can safely install 32-bit blender on 64-bit Windows.

==INSTALLATION==
* Copy the DLLs to the location where you installed blender
* Copy the PyPRP2 folder to your addons directory
   By default, this is %APPDATA%\Blender Foundation\Blender\2.57\scripts\addons
